This API provides Lufthansa's flight schedule information.

A search functionality is available based on a number of criteria:
  * airlines
  * flight number
  * start and end date
  * days of operation
  * origin and destination
  * aircraft types
  
Depending on access it enables to view passenger flights, cargo flights or both.

The flights returned are date-wise aggregated. 
Their structure and attributes follow the standards and definitions provided by the IATA Standard Schedules Information Manual (SSIM, Chapter 2).
